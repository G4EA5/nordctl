# Hooks

Copy scripts into `~/.config/nordctl/hooks/` — see [docs/HOOKS.md](../docs/HOOKS.md).

```bash
mkdir -p ~/.config/nordctl/hooks/post-preset
cp post-preset-default.sh ~/.config/nordctl/hooks/post-preset/default
chmod +x ~/.config/nordctl/hooks/post-preset/default
```
