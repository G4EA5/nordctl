Help & Setup

Everything you need to get nordctl working — from zero to streaming

Getting started

nordctl is a free, open-source helper for NordVPN on Linux. It does not include NordVPN itself — you install Nord’s official client once, log in, then use nordctl for one-click presets and a friendly web UI.

  1. Install nordctl: pip install nordctl or ./install.sh
  2. Initialize: nordctl init
  3. Start the UI: nordctl serve — open top bar Wizard or #dashboard/wizard for install and setup
  4. Install NordVPN (if using VPN presets): nordctl install-nordvpn
  5. Log in: nordvpn login
  6. Edit ~/.config/nordctl/config.yaml — or use the dashboard wizard / WiFi tab instead of manual YAML
  7. Preview or apply a preset: nordctl apply streaming-smartdns --dry-run
Tip: Hover any button in the dashboard for a short explanation. Use ↻ Refresh after changing settings in a terminal.

Installing NordVPN (official package)

nordctl cannot bundle NordVPN — that would violate Nord’s licence. Instead we run Nord Security’s official installer on Debian/Ubuntu, or show manual steps for other distros.

Automatic (Debian / Ubuntu / Mint / Pop!_OS)

nordctl install-nordvpn
# or from the dashboard: top bar Wizard → Install NordVPN (official)

This runs Nord’s apt repository setup and installs the nordvpn package. You will be prompted for your sudo password.

Manual steps (Debian/Ubuntu)

curl -fsSL https://repo.nordvpn.com/deb/nordvpn/debian/pool/main/n/nordvpn-release/nordvpn-release_1.0.0_all.deb -o /tmp/nordvpn-release.deb
sudo dpkg -i /tmp/nordvpn-release.deb
sudo apt update
sudo apt install -y nordvpn
sudo systemctl enable --now nordvpnd

Other Linux distros

Follow Nord’s official guide:

Fedora/RHEL, Arch, and others: install via Nord’s repo or AUR, then enable nordvpnd and run nordvpn login.

If install fails: read the error in the terminal or dashboard log. Common fixes:
  • dpkg/apt errors — run sudo apt --fix-broken install, then retry
  • curl not found — sudo apt install curl
  • Service won’t start — reboot once, then sudo systemctl start nordvpnd

Logging in to NordVPN

Login must be done in a terminal (not in the web UI — for your security).

nordvpn login

Follow the browser link or use a login token from Nord Account.

Verify:

nordvpn account
nordvpn status

Configure nordctl

Config file: ~/.config/nordctl/config.yaml

connect_country: United Kingdom

wifi:
  device: auto          # or wlan0 — auto-detect is fine
  profiles:
    - MyWiFi-5G
    - MyWiFi

smart_dns:
  primary: "103.86.96.103"
  secondary: "103.86.99.103"

server:
  bind: 127.0.0.1
  port: 8765

WiFi profile names

List your NetworkManager connection names (not SSID broadcast name — the saved profile name):

nmcli -t -f NAME connection show

Add every WiFi profile you use (2.4 GHz and 5 GHz often have separate names). Smart DNS presets update DNS on all listed profiles. These names also mark home WiFi for the top bar Home chip.

Home ISP in the top bar

With VPN on, nordctl shows your ISP address only on home WiFi (profiles above or trusted zones). Disconnect VPN once at home to auto-learn into ~/.config/nordctl/home_ip_cache.json, or set home_public_ip on a trusted zone. See Top bar IP.

wifi_zones:
  home_ip_learn: true
  home_ip_when_trusted: true
  trusted:
    - ssid: MyHomeWiFi
      preset: streaming-smartdns
      home_public_ip: 203.0.113.1

UI port

If port 8765 is busy, run nordctl init --fix-port. The server also auto-falls back at runtime and prints the URL.

In-app file editor

Click ✎ Edit files on the dashboard for a full-screen editor:

  • ~/.config/nordctl/config.yaml — main settings
  • ~/.config/nordctl/presets/*.yaml — your custom presets
  • Bundled presets — read-only reference

Features: YAML validate, WiFi profile picker, snippets, Ctrl+S to save, automatic backup (.bak) before each save.

Presets

Presets are one-click workflows: VPN connect/disconnect, Smart DNS, Meshnet, kill switch, split tunnel, and more. Click Apply on a preset card in the dashboard, or use the CLI:

nordctl presets
nordctl apply streaming-smartdns
nordctl apply meshnet-only

Some presets need connect_country or wifi.profiles in config — cards show “Needs config” when applicable.

Full catalog: see presets/README.md in the project repository.

TV streaming (Smart DNS)

  1. Enable Smart DNS in your Nord Account
  2. Add your public IP to the Smart DNS allowlist (the dashboard shows your IP when Smart DNS is active)
  3. Set wifi.profiles in config
  4. Apply preset TV streaming (Smart DNS) — VPN disconnects, Nord DNS is set on WiFi

Smart DNS does not encrypt traffic like VPN; it only routes DNS for region-unlocked streaming. Comply with your streaming service’s terms and local laws.

The top-right bar shows Home/Public → VPN · Mesh when available. Hover for details.

Travel-safe default

On hotel or café WiFi with VPN on, only VPN and Mesh show — your home ISP stays hidden.

Setup at home

  1. Add home connection names to wifi.profiles
  2. Disconnect VPN once on home WiFi (auto-learns ISP per SSID)
  3. Reconnect — top bar shows Home → VPN from cache

Optional: fixed ISP per SSID via wifi_zones.trusted[].home_public_ip in config.

Run the doctor

nordctl doctor

Each check includes fix steps. The dashboard Setup panel shows the same report.

nordvpnd not runningsudo systemctl enable --now nordvpnd
Smart DNS not stickingCheck wifi.profiles; may need sudo for system connections

FAQ

Does nordctl include NordVPN?
No. nordctl is MIT-licensed open source. NordVPN is installed separately from Nord Security’s official package.
Is this affiliated with NordVPN?
No. Independent tool. See LEGAL.md in the repository.
Why can’t I log in from the web UI?
Credentials should never go through a local web form. Use nordvpn login in your terminal.
Does install-nordvpn need internet?
Yes. It downloads Nord’s repository package and runs apt.
Why is Home IP missing with VPN on?
Home is shown only on home WiFi (wifi.profiles or trusted zones). Disconnect VPN once at home to auto-learn, or set home_public_ip on a zone. See Top bar IP.
macOS?
Not supported in v0.1 — Linux only (NetworkManager + nordvpn CLI).
