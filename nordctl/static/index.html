      •

      •

      •

                —
              
    

              
    
                Help
                
      
                  
    1. 1 Click Create a new preset below.
      2. 
                  
    2. 2 Give it a name and a one-line What it does (shown on the preset card — e.g. “Stream ITV — Smart DNS on WiFi”). Then pick country, Double VPN, Meshnet, Smart DNS, etc.
      3. 
                  
    3. 3 Read the help text under each option. Default / Don't include means skip. Grey rows are blocked — orange text explains why.
      4.

      5. 
                  
    5. 5 Click Save preset — it is saved to disk and listed under My presets on the My places tab.
      6. 
                  
    6. 6 Under Your presets below (or on My presets), click Edit preset to change every setting in the builder, then Save changes. You can also rename, Share, edit YAML, hide, or delete anytime.
      7. 
                  
    7. 7 Received YAML from someone? Paste it under Import a shared preset below (or use a raw URL).
      8. 
                
    
                
    Saving does not run the preset — you choose when to run it from My presets.

    Switches
    
                —
              
    
              
    Live NordVPN settings — one change at a time. Each card shows what is On/Off now, what it does, and yellow warnings before anything risky happens.
    

              
    
                Help
                
      
                  
    1. 1 VPN off — most switches change Nord settings for your next connect. Safe for kill switch, autoconnect, technology, etc.
      2. 
                  
    2. 2 VPN on — you are in an active session. Some switches are greyed out because they clash (e.g. OpenVPN UDP/TCP only when Technology is OpenVPN; Smart DNS on WiFi only when VPN is off).
      3. 
                  
    3. 3 Still clickable switches may reconnect or disconnect you — read the yellow box on the card and the confirm dialog before pressing.
      4. 
                  
    4. 4 Server groups (Onion, Double VPN, P2P…) show Active when you are already on that specialty server; other groups warn that Connect will switch servers.
      5. 
                  
    5. 5 Need several settings together (country + Meshnet + ports + DNS)? Use Create preset — one tap applies the full workflow instead of flipping switches one by one.

              Install baseline (rollback)
              —
              
                  
              On first use, nordctl automatically saves an install baseline — a backup of config, Wi‑Fi DNS, Nord settings, and IPv6 on this computer. Use this tab to restore that first-run state if something goes wrong.
              
                
              
              
              
              
              
              
              
              
              What gets restored
              
              
              
              
              
                
                
                
              
              
              
              
              
              
                
                
                
                
              
              
            
              
            
              
              
              
                
              
                  
              Factory reset
              —
              
                  
              Undo everything nordctl changed since install: restore the saved baseline, remove UI/tray services and schedule timers, clear logs and snapshots, and show the welcome screen again. Does not uninstall NordVPN or nordctl itself.
              
                
              
              
              
              
              
              
              
                Last resort only
                
              Factory reset is heavier than Rollback — it also clears activity logs, schedules, snapshots, and autostart services. NordVPN stays installed.
              
              
              
              
              
              
              
                
              
              
              
              Requires an install baseline (created on first nordctl init). Use Rollback for a lighter undo without clearing logs.
              
              
              
                
              
              
            
              
            
              
              
              
                
              
                  
              NordVPN snapshots
              —
              
                  
              Lightweight undo points for NordVPN settings only — useful when a preset changed kill switch, Meshnet, DNS, technology, or autoconnect and you want the previous Nord setup back in one click.
              
                
              
                
              
                  
                  
                
              
              
              
              
              
              
              
                
              
                  
              What is saved
              
                  
              A snapshot copies your current NordVPN CLI settings (from nordvpn settings) — kill switch, firewall, Meshnet, DNS, technology, and autoconnect. It does not include WiFi DNS, config.yaml, or IPv6.
              
                
              
                
              
                  
              When snapshots are created
              
                  
                
                    
              • Automatically before each preset run when auto_snapshot_before_preset is on (default).
                • 
                    
              • Manually — press Save snapshot before experimenting.
                • 
                  
              
                
              
              
              
              
              
                
              Which undo should I use?
              
                
                
                  
              • Restore latest snapshot (this page) — NordVPN settings only. Fastest fix for “that preset changed my Nord switches.”
                • 
                  
              • Rollback → Install baseline — restores config, WiFi DNS, Nord settings, IPv6, and more.
                • 
                  
              • Factory reset — baseline plus clearing logs, schedules, snapshots, and UI services.
                • 
                
              
              
              
              
              Saved snapshots
              
              
              Newest first. Restore latest always uses the most recent file (latest.json).
              
              
              
              
              
                
              
              
            
              
            
              
              
              NordVPN snapshots
              
              
              This tab stores quick undo points for NordVPN settings (kill switch, Meshnet, DNS, autoconnect, etc.). It works once the NordVPN CLI is installed and logged in.
              
              
              
                
              NordVPN required
              
                
              Snapshots are captured with nordvpn settings. Install NordVPN and log in from Setup, then return here to save or restore Nord-only undo points.
              
                
              If you need to undo WiFi DNS, Smart DNS, or config.yaml without Nord, use Rollback → Install baseline instead — that covers the full nordctl install state.
